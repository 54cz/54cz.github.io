Hexo+github搭建个人博客

发表于 2017-06-29

暑假开始了第一件事就是把博客搭建好,想着之后一边学习一边做笔记(以后回过头看的时候肯定很棒！)，说不定还能解决小伙伴们也遇到的问题，接下来进入正题开始搭建吧。

Hexo介绍

Hexo是一个基于Node.js的快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页，其生成的静态网页可以托管在Github上。

  • 超快速度
  • 支持MarkDown
  • 一键部署
  • 丰富的插件

创建github仓库

首先在github上注册一个账号，登录后在点击右上角头像旁边的加号选择New repository

Repository name 一定要填 你的账号名.github.io 例如：name.github.io 以后进入你的网站就是靠这个域名了
